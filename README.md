# 基于局域网的班级管理系统
这是SDU软件学院2018级的Java课设。  
本质上是一个mini版QQ，所以我在做的时候界面参考了TIM。

## 功能
- 低多边形控件
- 登录功能
- 注册功能
- 文件传输功能
- 1对1聊天、班级聊天（群聊）
- 公告功能
- 投票功能（老师的要求是随机传递，我跟老师讨论了一下改成了实时，投票为同意和不同意两个选项）
- 简单的基于Markdown的编辑器
